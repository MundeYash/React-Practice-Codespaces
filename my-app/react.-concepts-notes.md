This Repository covers important React concepts along with Definition and some examples

1.What is React and How does it work?

    `React ek javascript ki library h jo fast, interactive, re-usable UI components banane ke liye use hoti h`

    “React is a JavaScript library that helps us to  builds fast , interactive , re-usable UI using reusable components and efficiently updates the UI using Virtual DOM when state or props change.”

2.What is Jsx?

    `jsx (javascript XML)  ek syntax extension h jo javascript ke andar HTML like code likhne deta h`

    “JSX is a syntax extension for JavaScript that allows us to write HTML-like code inside JavaScript, which is later converted to React elements.”

3.What are Components?

    `Component React ka ek independent, reusable piece of UI hota h jo data accept karta h aur screen par output dikhata h`

    “Components are reusable UI blocks/components in react.”

4.What is difference between the class and functional Component?

    `functional component in react normal js function hota h jo jsx return karta h
     Class Components: class based components ek ES6 class hoti h jo React.component ko extend karti hai aur render() method return kari h`

    "Functional components are simple javascript functions and are preffered today, it return jsx
     Class Components : class components are older and uses lifecycle methods and extend react.component class and uses render() method."

5.What is Virtual DOM and how does react uses it?

    `Virtual DOM ek lightweight copy / JavaScript object hota hai jo Real DOM ki exact copy hoti hai. React isi copy pe changes karta hai pehle.`

    "Virtual DOM = Real DOM ka duplicate (copy)
     Virtual DOM is lightweight copy of actual DOM that React uses to efficiently update only the changed parts of the UI
    "
